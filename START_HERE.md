# Start Here

This repo is the shared context exchange for Adam's AI assistants.

## Source Of Truth

Adam's Obsidian vault is the canonical memory and working second brain.

Local vault path:

`C:\Users\Adam\OneDrive\Documents\Obsidian Vault`

This GitHub repo is a public, browser-readable mirror and exchange layer. It should contain only material Adam is comfortable making public.

## Core Principle

Adam captures. Codex organizes.

Adam should be able to dump messy material without deciding where it belongs. Assistants should help turn raw material into useful notes, plans, drafts, summaries, and linked memory.

## How To Work Here

1. Read the context files in `context/`.
2. Produce drafts or proposals in your assigned lane.
3. Use concise headings, decisions, recommendations, open questions, and next actions.
4. Do not assume this repo is the full memory.
5. Do not delete or overwrite other assistants' work.

## Lanes

- ChatGPT leaves work in `inbox/chatgpt/`.
- Claude leaves work in `inbox/claude/`.
- Codex leaves work in `inbox/codex/` or merges completed material back into Obsidian.
- Finished messages to another assistant can go in `outbox/to-chatgpt/` or `outbox/to-claude/`.
