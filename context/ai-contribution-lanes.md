# AI Contribution Lanes

## Purpose

This is the rule for how ChatGPT, Claude, Codex, Tabula, and other assistants should contribute without corrupting the source of truth.

## Source Of Truth

The Obsidian vault is the canonical memory.

This GitHub repo is a readable context mirror and exchange layer.

## Contribution Rule

Do not have multiple assistants edit the same canonical file at the same time. Each assistant should produce drafts, proposals, summaries, or structured outputs in its own lane.

## Recommended Repo Lanes

- `inbox/chatgpt/`
- `inbox/claude/`
- `inbox/codex/`
- `outbox/to-chatgpt/`
- `outbox/to-claude/`
- `proposals/`
- `archive/`

## Preferred Output Shape

When ChatGPT or Claude contributes, ask for:

- Summary
- Key claims
- Decisions or recommendations
- Open questions
- Suggested next actions
- Source links
- Anything that should be remembered long-term

## What To Avoid

- Giant unsorted docs.
- Multiple assistants editing the same memory page.
- Treating GitHub as the canonical second brain.
- Deleting source material without Adam's explicit approval.
