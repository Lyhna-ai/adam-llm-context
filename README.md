# Adam LLM Context Exchange

This repository is a public exchange layer for AI assistants working with Adam.

The source of truth is Adam's Obsidian vault. This repo is a readable context mirror and contribution inbox so ChatGPT, Claude, Codex, and other assistants can stay oriented without all editing the same private memory files.

## Contribution Rule

Do not rewrite canonical context files unless Adam explicitly asks.

Leave proposed updates in your own lane:

- ChatGPT: `inbox/chatgpt/`
- Claude: `inbox/claude/`
- Codex: `inbox/codex/`

Use one dated file per contribution:

`YYYY-MM-DD-short-topic.md`

Codex can review, merge, and file useful material back into Obsidian.
